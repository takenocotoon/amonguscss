Among us用カスタムCSSの書き換え補助ツール

Among usの色毎にDiscord IDを指定するタイプのカスタムCSSの書き換えをお手伝いするツールです。
「Input」にCSSを入力し下部の [Generate] ボタンを押すと、
「/*色名英語小文字ID*/」を対応するIDに書き換え、「Output」に出力します。

例）
img[src*="/*blackID*/"] → img[src*="123456789012345678"]

変換タグ一覧）
/*blueID*/ /*purpleID*/ /*redID*/ /*pinkID*/ /*orangeID*/ /*yellowID*/ /*limeID*/ /*greenID*/ /*cyanID*/ /*whiteID*/ /*blackID*/ /*brownID*/ /*tanID*/ /*coralID*/ /*bananaID*/ /*roseID*/ /*grayID*/ /*maroonID*/ /*noneID*/

Input



17-19桁の数字。Discordのユーザー設定→詳細設定→開発者モードを有効→ユーザーを右クリック→IDをコピー。



Output


テスト用ページ

生成したCSSとセットで使えます。配信ソフトに読み込ませてCSSのテストや位置調整にお使い下さい。
15人まで対応しています。
※ユーザーアイコンにサンプル用としていらすとやさんの画像を使っています。


サンプル


AmongUsCapture連携（テスト中）


テスト中の機能です。
AmongUsCaptureと連携し、自動で色設定をします。
Discordの名前とAmongUs内の名前が違う場合は↑から名前の登録をしておいて下さい。

AmongUsCapture起動 → 設定からAPIサーバーをオン → ↓の連携開始をクリック
→ AmongUs起動 → ロビーへ行く → 色の反映を確認できたら↑のSetをクリック、CSSをコピーしてOBSに適応

※自動で行われるのは色の選択のみで、CSSの生成とOBSへのカスタムCSS適応は手動で行う必要があります。
※起動の順番が前後すると正常に動かないことがあります。

このページで入力したIDなどの情報はお使いのブラウザのローカルストレージに保存されます。サーバには一切の情報を送信していません。
